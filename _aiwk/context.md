# Context

## Project Overview
This is Vikas Bhatia's personal portfolio website showcasing cybersecurity executive expertise. Built with React 18, TypeScript, Vite, and Tailwind CSS.

## Current Major Initiative
**AI-Powered Lead Generation & Business Transformation Platform**

A multi-business lead generation platform to capture inbound leads for four virtual service businesses (Handyman, AV/Tech, Pool Care, Garage Organization), qualify them via AI voice/text agents, and hand them off to real businesses as proof of AI transformation capabilities.

### Project Status
- **Phase:** Planning Complete, Ready for Implementation
- **Implementation Plan:** Created comprehensive plan in `_aiwk/plan/implementation-plan.md`
- **Timeline:** 8-12 weeks estimated
- **Next Steps:** Begin Phase 1 (Foundation & Database)

### Key Components
1. **Frontend:** 4 landing pages with lead capture forms
2. **Backend:** Supabase database with Edge Functions
3. **AI Agents:** Lindy.ai for inbound/outbound voice and SMS
4. **Automation:** n8n workflows for lead processing and handoff
5. **Prospecting:** Automated business discovery and outreach system
